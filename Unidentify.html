Unidentify Report

¿Cómo obtener el archivo en Oracle?

1

Exportar desde Oracle

Desde Oracle, se debe ejecutar el reporte desde view - Find - Escoge las fechas y selecciona StateUnidentify click en Find.

2

Descargar reporte desde File

En el momento que descarga, vamos a File - export.

3

Abre el archivo que descarga Oracle

En la columna Receipt Number pasala a texto y guarda el archivo en excel.

4

Sube el archivo de excel

Sube el archivo .xls generado, en la caja de esta plataforma.

Sube tu archivo de hoja de cálculo (.xls/.xlsx) para generar el Unidentify por cada Banco en formato Excel.
