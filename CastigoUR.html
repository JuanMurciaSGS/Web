Castigo Unbill Revenue

Descarga reporte en Oracle

1. Desde Oracle, se debe ejecutar el reporte con fecha del día: Request - Submit New Request - SGS Order Lines Summary Report -

Vamos a descargar 3 reportes con fechas desde el año 2023, tal como se ve en la imagen. Primero 2023, luego 2024 y luego 2025. Tener en cuenta que este proceso puede tardar entre 3 o 4 horas. Se debe correr el reporte en el momento de la confirmación de cierre de Billing.

2. En el momento que descarga, vamos a Actions - Zip and Download Output, el sistema descarga una carpeta comprimida, y en esta hay un archivo de excel.

Vamos a descargar los 3 reportes y lo vamos a guardar en la carpeta Python - Castigo Unbill Revenue - Castigo UR. Esto con el fin de realizar el analisis de datos, la limpieza, trasformación y extracción de lo que necesitamos para hacer el castigo. Esto nos asegura tener un archivo limpio y manejable para hacer los cruces de información.

3. Traemos el archivo del Oct aprobado para el cierre.

Vamos a copiar el archivo del Otc y lo vamos a guardar en la carpeta Python - Castigo Unbill Revenue - OTC. Esto con el fin de realizar el analisis de datos, la limpieza, trasformación y extracción de lo que necesitamos para hacer el castigo. Esto nos asegura tener un archivo limpio y manejable para hacer los cruces de información.

4. El primer script de Python que debemos correr es script_consolidar

Al ejecutar este Script nos va a generar un archivo de excel llamado Reporte_Consolidado_Final que vamos a dejar en la carpeta.

5. El segundo script de Python que debemos correr es script_otc

Al ejecutar este Script nos va a generar un archivo de excel llamado Reporte_OTC_Concatenado que vamos a dejar en la carpeta.

6. El tercer script de Python que debemos correr es script_cruce

Al ejecutar este Script nos va a generar un archivo de excel llamado Reporte_Final_CRUZADO_Y_CALCULADO_FILTRADO que vamos a dejar en la carpeta.

7. El reporte que genera es el final del castigo calculado. Listo para enviar a su aprobación.
