Descarga reporte en Oracle desde el Gl User

1. Desde Oracle, buscar la opción Launch Journal Wizard.

2. Nos abre una pagina web donde vamos a seleccionar el Layout que necesitamos para el reporte. Tener en cuenta los mas comunes son: SGS Functional Actuals - Single Extended es para Asientos en Soles, el SGS Foreign Actuals - Single Extended es para asientos en Dolares

3. El sistema nos va a descargar un archivo en Excel en el cual debemos trabajar. La primera acción es ingresar a Archivo - Información - y Desproteger.

4. Vamos a seleccionar dando doble click sobre cada uno de los campos obligatorios.

5. La categoria que vamos a escoger depende del tipo de Asiento que necesitamos. Si queremos que el asiento se mantenga en el tiempo se debe escogeer una Catagoria Accrual (ejemplo para INGRESOS DIFERIDOS). Si queremos que el asiento se contabilice en el mes pero al siguiente se reverse, se debe escoger catagoria Accrual Autoreverse (ejemplo para Asientos Manuales, Bad Debt, Castigo UR). Si queremos hacer una reclasificación debemos escoger categoria Reclass.

6. El nombre del asiento si es un Asiento Manual, se debe llamar como el siguiente ejemplo, primero debe decir Manual, luego las iniciales de la persona que crea el asiento, el Nombre del asiento y por ultimo el periodo. "Manual_JM_INGRESOS DIFERIDOS_JAN-26

7. Debes rellenar la información del asiento segun las columnas descritas a continuación, tener presente que si necesitan mas lineas solo deben agregar filas hasta que tengan la capacidad para copiar el asiento completo. Ojo que las filas de los totales deben quedar libres para que el sistema haga la validación de sumasn iguales en el Credit y en el Debit

8. Cuando el asiento este completo se debe subir a Oracle con el siguiente boton. Y seguir las instrucciones de la confirmación del sistema.

9. Cuando el asiento cargue correctamente debes volver a Oracle en el boton Enter Journal. Vas a buscar con los parametros del Period y Status Unposted para buscar el asiento recien creado.

10. Cuando se ubique el asiento se ingresa por el boton Review Journal. Se debe confirmar la información que sea el asiento.

11. No olvidar cargar el sustento en el Clip. Y si esta listo enviar a Aprobar y luego Postear.

12. Si el asiento necesita niveles de aprobación, estas le llegaran a los responsables y hasta que se cumpla el flujo lo puedes postear.
