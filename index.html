Ageing Report

¿Cómo obtener el archivo en Oracle?

1

Exportar desde Oracle

Desde Oracle, se debe ejecutar el reporte con fecha del día: Request - Submit New Request - SGS Ageing Detail RetroActive TEXT (ALL) Con los parametros de la imagen.

2

Descargar Carpeta comprimida con el Archivo

En el momento que descarga, vamos a Actions - Zip and Download Output, el sistema descarga una carpeta comprimida, y en esta hay un archivo de texto.

3

Sube el archivo de texto de la carpeta

Sube el archivo .txt generado en la caja de esta plataforma.

4

Validar Excel

Haz clic en "Generar Excel" y verifica que los ageing o informes de cierre coincidan con tu operación.

Sube tu archivo de texto (.txt) para generar el Ageing por cada FCODE en formato Excel (.csv).
